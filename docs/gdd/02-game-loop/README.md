# Game Loop & Turn Structure

## Campaign Structure

A single roguelike run is called a **campaign**. It spans from character creation through career end (win, loss, or retirement).

```
Character Creation → Weekly Turns → ... → Career End
```

## Weekly Turn Phases

### 1. Dawn Phase (read-only, automated)

The world advances before the player acts:

- World simulation ticks (economy, demographics, geopolitics)
- NPC autonomous actions resolve
- Morning briefing generated by Gemma 4
- Social graph propagation (rumors, reputation shifts)
- News cycle advances

### 2. Action Phase (player spends Action Points)

Player's core gameplay loop:

- Base AP pool varies by office level + traits + cards
- Actions: meet NPC, give speech, draft legislation, campaign, fundraise, research, play tactic cards, etc.
- Each action costs AP and may trigger dice rolls
- Conversations with NPCs are free once entered (entry costs AP)
- NPCs who approach YOU are always free
- Phone calls cost less AP than in-person meetings

### 3. Event Phase (interrupt-driven)

Random or triggered events fire:

- Some events are mandatory (floor votes, court hearings)
- Special event phases have their own sub-mechanics:

| Event Type | Mechanics |
|-----------|-----------|
| Debate | Real-time card play + live dialogue, timed rounds |
| Crisis | Time-limited decisions, cascading consequences |
| Election | Multi-week campaign arc, polls, rallies |
| Trial/Hearing | Structured Q&A, evidence presentation |
| Negotiation | Multi-party deal-making, trade-offs |
| Press Conference | Q&A rounds, spin cards |
| Floor Vote | Whip phase then sequential vote |

### 4. Dusk Phase (resolution, automated)

Turn wraps up:

- Action consequences resolve
- Relationship graph updates
- Position card coherence check (flip-flop risk evaluation)
- Score/reputation tallied
- Autosave to RocksDB snapshot

## Action Point Economy

| Office Level | Base AP | Examples |
|-------------|---------|----------|
| Local | 5 | City council, school board |
| State | 6-7 | State legislator, mayor |
| Federal | 8-10 | US House, Senate |
| Executive | 10-12 | Governor, President |
| Bureaucratic | 5-8 | Varies by rank/role |

Modifiers:
- Trait cards: "Workaholic" +2 AP, "Health Issues" -1 AP
- Asset cards: domain-specific free actions (media connections = free PR)
- Staff: good scheduler = +1 AP
- Age/stress: penalties at high levels
- **Unspent AP do NOT carry over** — use it or lose it

## Dice Roll System

D20-based with modifiers:

### Roll Categories
- Persuasion
- Intimidation
- Deception
- Policy Knowledge
- Media Savvy
- Negotiation
- Luck
- Cunning
- Analysis
- Discretion
- Endurance

### Difficulty Classes

Gemma 4 sets DC based on narrative context:

| DC | Difficulty | Example |
|----|-----------|---------|
| 5 | Trivial | Convince an ally to support your bill |
| 8 | Easy | Friendly NPC, simple request |
| 12 | Moderate | Neutral NPC, reasonable request |
| 16 | Hard | Skeptical NPC, significant ask |
| 18 | Very Hard | Hostile NPC, major concession |
| 20+ | Near Impossible | Enemies, outrageous requests |

### Roll Resolution

```
result = d20 + skill_modifier + card_bonuses + situational_modifiers
```

- **Critical success (nat 20)**: Triggers special narrative branches
- **Critical failure (nat 1)**: Triggers negative narrative branches
- Cards can modify rolls: "+3 to Persuasion when addressing labor unions"
- Relationship modifiers apply: high trust = lower DC

## Phase Rigidity Spectrum

Phases range from fully freeform to structured. **Even in structured phases, the player can ALWAYS type freeform.** The DM interprets it — with consequences.

```
FREEFORM ◄──────────────────────────────────────────► STRUCTURED

Free Roam       Conversations    Press Conf      Floor Vote
Downtime        Negotiations     Debate          Election Night
Drafting Law    Campaign Events  Court Hearing   Crisis (late)
```

Examples of going off-script:
- Walk out of a debate → media firestorm, poll impact
- Ignore a crisis to go golfing → Channel 4 has the footage
- Interrupt a floor vote to confront a senator → miss your chance to speak

## Special Event Phases

### Debate
- Timed rounds (configurable timer or untimed)
- Play tactic cards each round for modifier
- Gemma 4 judges effectiveness, generates crowd reaction
- Scoring system: cumulative points across rounds
- Audience mood tracked

### Crisis
- Free to enter (mandatory)
- Limited number of decisions
- Cascading consequences tracked across future weeks
- Urgency indicator in UI
- Some crises have escalation tracks

### Election
- Multi-week arc (see [Elections](../19-elections/README.md))
- Resource allocation across districts
- Separate AP budget for campaign actions

### Floor Vote
- Whip phase: spend AP lobbying colleagues before vote
- Vote resolution: each NPC votes based on ideology, party pressure, relationship, constituent pressure, deals made
- Results tallied in real-time display
